Wijmo FlexGrid is a framework agnostic, Excel-like datagrid with zero dependencies. This overview will detail stand-out features such as Grouping, Pagination, Filtering, Pivot Tables, and more.

#### Features Overview

* Sorting
* Filtering
* Searching
* Merging
* Paging
* Pinning
* Selection
* Cell Templates
* Editing
* Grouping
* Aggregation
* TreeGrid
* Pivot Tables
* Master Detail
* Import/Export
* Globalization
